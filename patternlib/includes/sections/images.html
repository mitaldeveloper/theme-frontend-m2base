Outset

The outset utility can be used to outset an image to either the left, right or both sides of its container.

.o-outset--left
.o-outset--right
.o-outset--both

Image Breakout

An alternative to o-outset (which only outsets images by a fixed amount) is o-breakout. This breaks an image out of its parent wrapper to the full width of the viewport on both sides. To use it wrap an image in a div tag with the class o-breakout.

o-breakout

Image Cover

Covers the background of an element with an image. It uses the background-size: cover property, so the image will be rendered, preserving its intrinsic proportion, at the largest size contained within, or covering, the background positioning area.

Usage:

Define the image source inline in the style attribute of the element. The element needs to be given some dimensions, so this object works well with the o-ratio-embed object.

Example markup:

<div class="o-img-cover" style="background-image: url(/static/frontend/GPMD/M2base/en_GB/images/patternlib/patternlib-19.jpg);"></div>
